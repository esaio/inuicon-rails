# inuicon-rails

Icon web font for Rails (used in pplog.net) https://taea.github.io/inuicon/demo.html

## Installation

1) Add this to your `Gemfile`:

```ruby
# Assets
gem 'inuicon-rails'
```

2) Add this to your Sass file:

```sass
@import inuicon
```

## Usage

- [Inuicon Sample](https://taea.github.io/inuicon/demo.html)
